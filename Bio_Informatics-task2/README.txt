Below are the steps to be followed to run the program:

1. Copy the submitted folder and save it.
2. Log in to fast.cs.odu.edu with password credentials.
3. Go to the folder where the file is saved, there is a makefile created already.
4. Enter make, this create the executable.
5. To run the program type ./extractdata.
6. It will ask to input the pdb file, please enter the pdb filename with extension.
7. After you click on enter, the pdb files get generated with the name Aligneddata.pdb in the current folder.
8. You can view the data using Notepad++ or chimera.
